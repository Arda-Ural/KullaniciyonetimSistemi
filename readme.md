
# Kullanıcı Yönetim Sistemi (Console CRUD)

Bu proje, C# dili kullanılarak geliştirilmiş, rol tabanlı bir kullanıcı yönetim simülasyonudur.

### Özellikler:
- **Admin & User Rolleri:** Yetkiye dayalı menü gösterimi.
- **Full CRUD:** Kullanıcı ekleme, listeleme, güncelleme (yetki verme) ve silme.
- **Güvenli Giriş:** Kullanıcı adı ve şifre doğrulaması.
- **Dinamik Bellek:** Veriler program çalıştığı sürece `List<User>` üzerinde tutulur.

### Nasıl Çalıştırılır?
`Program.cs` dosyasını herhangi bir C# derleyicisinde (Visual Studio veya OnlineGDB) çalıştırabilirsiniz. 
**Varsayılan Admin Bilgileri:** 
- Kullanıcı Adı: `admin` 
- Şifre: `123`
